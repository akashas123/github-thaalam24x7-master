# Thaalam 24x7 – Android App

Thaalam 24x7 is a live online radio Android application built using native Android (Kotlin).

## Features
- Live radio streaming
- Media notification controls
- Background playback
- Lightweight WebView-based UI

## Tech Stack
- API Endpoints
- Oracle Cloud Infrastructure
- Kotlin
- Android Native Framework

## APK
The installable APK is available in the project folder.
